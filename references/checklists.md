# 执行检查清单

## 举一反三检查清单

- [ ] 分析了问题的根本原因
- [ ] 抽象出通用模式/规律
- [ ] 检查了其他可能受影响的领域
- [ ] 批量修复了同类问题

## 事不过三检查清单

- [ ] 第1次尝试: 直接方法
- [ ] 第2次尝试: 优化调整
- [ ] 第3次尝试: 彻底改变策略
- [ ] 超过3次则必须停止并寻求帮助

## 大事化小检查清单

- [ ] 明确了最终目标
- [ ] 按维度/模块/阶段拆分
- [ ] 确定了优先级和依赖关系
- [ ] 制定了逐个击破计划

## 小事化了检查清单

- [ ] 找到了根本原因
- [ ] 修复方案彻底，无临时方案
- [ ] 验证了修复有效
- [ ] 添加了监控/预防措施
- [ ] 经验已记录分享
